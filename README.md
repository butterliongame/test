# GitHub Pages 網站

這是一個可直接部署到 GitHub Pages 的靜態網站。

## 檔案

- `index.html`：網站內容
- `styles.css`：版面與視覺樣式
- `script.js`：手機選單互動
- `assets/hero-dashboard.png`：首頁主視覺

## 用 GitHub 網頁部署

1. 到 GitHub 建立新的 repository。
2. 上傳這個資料夾內的所有檔案。
3. 進入 repository 的 `Settings`。
4. 左側選 `Pages`。
5. 在 `Build and deployment` 選擇 `Deploy from a branch`。
6. Branch 選 `main`，資料夾選 `/root`。
7. 儲存後等待幾分鐘，GitHub 會顯示你的網站網址。

## 建議替換

- 將 `你的品牌` 改成你的網站名稱。
- 將 `hello@example.com` 改成你的聯絡信箱。
- 修改作品、服務與流程文案。
- 如果 repository 名稱不是 `username.github.io`，網站網址會像 `https://username.github.io/repository-name/`。
